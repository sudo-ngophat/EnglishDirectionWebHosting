Level 1 — Comparatives

So sánh hơn (Comparatives)

Dùng để so sánh 2 thứ với nhau.

Từ ngắn (1 âm tiết): adj + -er + than
   tall → taller than / fast → faster than / old → older than

Từ dài (2+ âm tiết): more + adj + than
   expensive → more expensive than / beautiful → more beautiful than

Bất quy tắc:
   good → better / bad → worse / far → farther/further
Quy tắc thêm -er:
• Hầu hết: thêm -er (taller, faster, older)
• Kết thúc -e: thêm -r (nicer, larger)
• 1 nguyên âm + 1 phụ âm: gấp đôi (bigger, hotter, thinner)
• Kết thúc -y: đổi y → ier (happier, easier, busier)
• 2 âm tiết kết thúc -y, -ow, -er: dùng -er (happier, narrower, cleverer)

Bài 1: Điền dạng so sánh hơn (20 câu)

Bài 2: Chọn đáp án đúng (10 câu)
