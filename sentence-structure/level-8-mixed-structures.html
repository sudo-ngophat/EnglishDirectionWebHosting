Level 8 — Mixed BOSS

Tổng hợp tất cả cấu trúc câu

Simple: S + V + O (1 mệnh đề)
Compound: MC + , FANBOYS + MC (and/but/or/so/yet)
Complex: MC + SC (because/although/when/if/while...)
Relative Clause: ...noun + who/which/that/whose/where...
Passive: S + be + V3 (+ by agent)
Conditional: If + condition, result (Type 0/1/2/3)
Reported: He said (that) + lùi thì
Chiến lược:
1. Xác định có bao nhiêu mệnh đề?
2. Mệnh đề nối bằng gì? (FANBOYS = ghép, subordinator = phức)
3. Có bị động không? (be + V3)
4. Có điều kiện không? (If...)
5. Có tường thuật không? (said/told/asked + lùi thì)

Bài 1: Xác định loại cấu trúc (15 câu)

Bài 2: Chuyển đổi cấu trúc câu (12 câu)

Bài 3: Sửa lỗi sai — Tổng hợp (10 câu)

Bài 4: Kết hợp nhiều cấu trúc trong 1 câu (8 câu)

Nối các ý thành 1 câu hoàn chỉnh dùng cấu trúc được yêu cầu.
